# Site Log

Offline site log for construction lots. One HTML file, no internet needed, no account.
Everything saves to the phone browser's local storage.

## Get it on your iPhone

1. **Send `sitelog.html` to your phone** — email it to yourself, or put it in
   iCloud Drive / Google Drive, then save it into the **Files** app.

2. **Files' preview won't run it** — it has to open in Safari. The share sheet
   often hides "Open in Safari" for HTML files, so use the Shortcuts route:

   - Open the **Shortcuts** app → **+** to make a new shortcut
   - Add action **Get File** → pick `sitelog.html` → turn **Show Document Picker** OFF
   - Add action **Open in Safari** (do NOT use Quick Look — it won't run the app)
   - Name it "Site Log" and **Add to Home Screen** (shortcut settings → share → Add to Home Screen)

3. Tap the Home Screen icon — the app opens in Safari, fully offline.

## First-time setup

1. Go to the **Lots** tab and add your 3–5 active addresses
2. Use **New Entry** on-site: pick the lot, tap weather + trades chips, snap photos, add a note — ~30 seconds
3. **Export** tab builds a clean text summary (by lot / date range) to copy or share to the super

## ⚠️ Back up weekly — not optional

iOS can silently wipe browser storage for file-based pages (storage pressure, or
~7 days without opening the app). If that happens your entries are gone **unless
you have a backup**.

- **Export tab → "Back up to file"** — saves a `sitelog-backup-YYYY-MM-DD.json` into Files (photos included)
- **"Restore from backup"** brings everything back in seconds (choose replace or merge)
- Do it at the end of every week. The Export tab shows a storage meter — if it turns red, back up and delete old entries.
